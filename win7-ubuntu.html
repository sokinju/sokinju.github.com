从win7连接ubuntu

Thu 21 June 2012
By soki

In 学习.

tags: ubuntu，win7，远程桌面链接

搞了一个下午，试过vnc 和 XManager，还是这个最简单好用。嘻嘻~~~

首先，我使用的win7，ubuntu10.10的操作系统。下面要从win7远程连接到ubuntu，步骤如下：

ubuntu下： 1、Win7远程连接上Ubuntu，所使用的协议是rdp，所以我们要装这个东西。打开终端：

 sudo apt-get install xrdp

2、装上xrdp之后，再：

 sudo apt-get install vnc4server tightvncserver

3、都装上了之后，在“System Setting—>Remote Desktop”那里，设置好,允许远程桌面，允许控制，等等。
